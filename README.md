# Forseen (YHack-2026)

Regulatory analysis demo: **React** frontend, **FastAPI** backend, **K2** + **Hermes** model services, and **MongoDB** for signals. Company setup, dashboard, analysis flow, and RAG chat.

## Run locally

```bash
./scripts/run-local.sh
```

- Frontend: [http://127.0.0.1:5175](http://127.0.0.1:5175)  
- API: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)  

Stop: `./scripts/stop-local.sh`
